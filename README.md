# Quantum Fourier Transform App

An interactive **Streamlit application** showcasing the **Quantum Fourier Transform (QFT)** using variational quantum circuits built with [PennyLane](https://pennylane.ai/).

---

## 🚀 Key Features
- **Demo Mode**: Run experiments instantly with the Iris dataset.
- **Upload Mode**: Bring your own dataset (CSV) for custom runs.
- **Sample Dataset**: Download a ready‑made CSV to test the app.
- **Validation & Error Handling**: Professional feedback if datasets are invalid.
- **Optimization Visualization**: Plot cost vs. iterations and view optimized parameters.
